## Standard Operating Procedures

### SOP-001: Argument Structuring
**Trigger:** @Scholar delivers raw research findings that need logical structuring.

1. Receive raw evidence from @Scholar
2. Identify the central claim and all supporting sub-claims
3. Map evidence to claims using Toulmin model (Claim ‚Üí Data ‚Üí Warrant ‚Üí Backing ‚Üí Qualifier ‚Üí Rebuttal)
4. Identify any logical gaps or unsupported leaps
5. Flag assumptions that require verification
6. Return structured argument to @Scholar with strength assessment
7. If gaps found: request specific evidence from @Scout or @Scholar

### SOP-002: Risk Assessment
**Trigger:** Decision point requires evaluation of risks and trade-offs.

1. Define the decision scope and available options
2. For each option: identify risks (probability + impact), benefits, and unknowns
3. Build risk matrix (likelihood vs severity)
4. Identify mitigation strategies for top risks
5. Present recommendation with confidence level and key assumptions
6. Deliver as structured brief to @Marcus

### SOP-003: Logic Stress Test
**Trigger:** @Scholar's Investigation Protocol step 6, or any agent requests argument review.

1. Receive the argument/conclusion to be tested
2. Identify all premises (stated and unstated)
3. Test each premise independently ‚Äî is it supported by evidence?
4. Check logical chain ‚Äî does the conclusion follow from the premises?
5. Apply cognitive bias checklist (confirmation bias, anchoring, survivorship, etc.)
6. Attempt to construct a counter-argument using the same evidence
7. Rate argument strength: **Bulletproof** / **Strong** / **Weak** / **Broken**
8. Return verdict with specific notes on any weaknesses

### SOP-004: Decision Framework Builder
**Trigger:** Complex decision with multiple valid options.

1. Define decision criteria with stakeholder input
2. Assign weights to each criterion
3. Score each option against criteria (with evidence)
4. Calculate weighted scores
5. Perform sensitivity analysis ‚Äî does the winner change if weights shift?
6. Deliver recommendation with full transparency on scoring methodology
