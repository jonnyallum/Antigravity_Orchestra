### SOP-002: Page Copy Optimization
**Trigger:** New page being built, or existing page needs copy refresh.

1. Read the page's purpose and conversion goal
2. Check brand voice profile for this client
3. Write copy following the hierarchy:
   ```
   H1: The Promise (what they get)
   Subhead: The Proof (why believe us)
   Body: The Path (how it works)
   CTA: The Push (what to do next)
   ```

### SOP-003: Copy Gate Sign-Off (NEW)
**Trigger:** Page or feature ready for final copy review.

**Copy Gate Checklist:**
- [ ] No placeholder text (Lorem ipsum = instant fail)
- [ ] No generic phrases ("Welcome to our website")
- [ ] All CTAs are specific and action-oriented ("Start your free quiz" not "Click here")
- [ ] Brand voice is consistent with client profile
- [ ] H1 is unique and compelling (not repeated across pages)
- [ ] Social proof is real (not fabricated testimonials)
- [ ] Error messages are helpful and human
- [ ] Empty states have personality
- [ ] Loading states have microcopy
- [ ] 404 page has brand-appropriate humor

**Sign-off statement:** "Copy verified. Voice consistent. No ghosts. No placeholders. â€” @Elena"

### SOP-005: CTA Engineering (NEW)
**Trigger:** Any page with a conversion goal.

**CTA Framework:**
| Type | Pattern | Example |
|:-----|:--------|:--------|
| Primary | Action + Benefit | "Start your free quiz" |
| Secondary | Lower commitment | "See how it works" |
| Urgency | Time pressure | "Limited spots this month" |
| Social | Peer pressure | "Join 500+ quiz hosts" |

**Rules:**
1. Every page has exactly ONE primary CTA
2. CTA button text is a verb phrase (never "Submit" or "Click here")
3. CTA is visible without scrolling on desktop
4. Surrounding copy reduces anxiety ("No credit card required")
5. After-click copy confirms the action ("You're in! Check your email")

### SOP-006: Microcopy Library (NEW)
**Trigger:** UI component needs text (buttons, errors, empty states, tooltips).

### SOP-007: Content Audit (NEW)
**Trigger:** @Vigil flags copy quality issues, or quarterly review.

1. Crawl all pages of the target project
2. Flag violations:
   - Placeholder text (Latin, "lorem ipsum," "TBD")

   - Missing microcopy (empty states, error messages)
3. Score each page: 1-10 for copy quality
4. Create remediation plan with priority order
5. Report to @Marcus and @Rowan
