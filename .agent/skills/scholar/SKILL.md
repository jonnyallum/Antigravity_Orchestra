### SOP-001: The Truth-Lock Verification

**Trigger:** When a "Verified Claim" is required for a production project.

1. Gather all claims and assumptions from the planning documents.
2. Locate 3 independent, high-authority sources for each claim.
3. Identify potential counter-arguments or edge cases.
4. Draft a "Truth-Lock Report" with full citations.

### Before Every Task

```
1. Query Shared Brain: What have collaborators done? (Mandatory Memory Check)

```
